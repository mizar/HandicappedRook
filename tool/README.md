## レーティング推定器 (infer_rating.py)

厳密な方法で視覚的にわかりやすいのがないので作った。

- Python 3.6 以降、 [SciPy](https://www.scipy.org/) が実行に必要。
- 引数は順に勝ち数、負け数、引き分け数(デフォルト=0)。
